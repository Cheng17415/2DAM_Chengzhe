Para que los ejercicios que utilizan JPA funcionen correctamente, se necesita:
- MySQL WorkBench
- MySQL Server
- mysql-connector-j.8.0.33.jar (Se encuentra en esta carpeta)
- [hibernate](https://hibernate.org/orm/releases/5.4)
  1. Download zip
  2. Descomprimir
  3. Abrir Eclipse, en la carpeta del ejercicio con JPA:
     click derecho -> Build Path -> Add Libraries -> User Library
     -> User Libraries -> New -> hibernate -> OK -> Add External Jars
     -> Select all of hibernate/lib/required.
